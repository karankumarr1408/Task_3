# Simple Quiz Application

## Description
A Java-based Quiz Application that displays multiple-choice questions, accepts user answers, and calculates the final score.

## Features
- Multiple-choice questions
- User answer input
- Score calculation
- Uses ArrayList and HashMap
- Console-based interface

## Technologies Used
- Java
- ArrayList
- HashMap
- OOP

## How to Run
```bash
javac *.java
java Main
```
